Programación Web

HTML

Es un lenguaje utilizado para el desarrollo de paginas web.
Podemos utilizar cualquier editor de texto para manipular HTML.

JavaScript

Es un lenguaje que nos permite crear pequeños programas que se insertan en paginas web.
JavaScript nos permite crear efectos dinamicos para nuestras paginas web.

JQUERY

Son librerias y ficheros de javascript que nos facilita ejecutar tareas en paginas web.
JQUERY nos permite implementar procesos ya listos para usar, facilitandonnos la solucion de tareas basicas.
